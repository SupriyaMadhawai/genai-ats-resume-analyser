## AI ATS Resume Analyzer

🔍 Overview
An AI-powered ATS system that evaluates resumes against job descriptions using Google Gemini.

🚀 Features
- Upload PDF resume
- Job description matching
- ATS score generation
- AI-based suggestions

🛠 Tech Stack
- Python
- LangChain
- Google Gemini API
- Streamlit

🌐 Live Demo
(https://genai-ats-resume-analyser-fjygepngcykrlbgzwcvojc.streamlit.app/)
📦 Installation
pip install -r requirements.txt
streamlit run app.py
